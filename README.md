# QWERhacks EdTech Dashboard (LearnHub)

Full-stack EdTech dashboard with Next.js (App Router) + Express + MongoDB Atlas + JWT auth in httpOnly cookies.

---

## Tech Stack

### Frontend
- Next.js (App Router)
- TypeScript
- shadcn/ui + lucide-react
- Cookie-based auth (`credentials: "include"`)

### Backend
- Express (`http://localhost:4000`)
- MongoDB Atlas + Mongoose
- Auth: JWT in httpOnly cookie named `token`
- bcrypt password hashing
- cookie-parser enabled
- CORS configured for cookies
